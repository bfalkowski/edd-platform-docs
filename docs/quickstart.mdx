This repository currently contains documentation for the EDD Platform project. The docs are built with Mintlify and live under `docs/`.

## Preview the docs

Run Mintlify from the documentation root:

```bash
cd docs
npx mint dev
```

Mintlify usually serves the local preview at `http://localhost:3000`.

## Read the project in order

1. Start with [Eval-Driven Design](/concepts/eval-driven-design) to understand the product thesis.
2. Read [Architecture overview](/architecture/overview) to see the workflow/control plane model.
3. Use the [Glossary](/reference/glossary) to keep terminology consistent.
4. Review [Local development](/guides/local-development) before running project commands.

## What not to assume

This docs site describes both current repository structure and intended product design. When a page describes planned workflow behavior, it uses language such as "intended design" or "should." Do not treat those sections as proof that a production implementation exists.
